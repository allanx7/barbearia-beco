Beco Barbearia

Landing Page desenvolvida para a Beco Barbearia, 
com o objetivo de apresentar os serviços da barbearia e 
facilitar o contato com os clientes.

🚀 Tecnologias

- HTML5
- CSS3

✨ Funcionalidades

- Layout responsivo
- Design moderno
- Seção de serviços
- Galeria de imagens
- Botão para contato via WhatsApp
- Informações de localização e horário de atendimento

🎯 Objetivo

Criar uma presença online profissional para a Beco Barbearia, proporcionando uma experiência simples, rápida e intuitiva aos clientes.
